# SG Indicative Valuation Tool

A machine learning-powered indicative valuation tool for Singapore non-landed residential properties, built on URA caveat data.

Live app: [sgvaluation.streamlit.app](https://sgvaluation.streamlit.app)

---

## What It Does

- Enter a property address and unit number
- The app looks up property details via OneMap API
- Runs 8 Random Forest models (Resale + New Sale × 4 time periods)
- Returns a blended indicative value with 95% confidence interval
- Shows comparable transactions from the same period
- Allows valuers to submit their own assessment, which is saved to Google Sheets

---

## Model Architecture

| Sale Type | Periods | Model Files |
|---|---|---|
| Resale | 6m, 1y, 18m, 3y | `rf_model_resale_*.pkl` |
| New Sale / Sub Sale | 6m, 1y, 18m, 3y | `rf_model_newsale_*.pkl` |

**Split:** 40% Train / 30% Validation / 30% Test

**Features used:**
- Project Name, Type of Area, Property Type
- Planning Region, Planning Area, Market Segment
- Tenure Group, Completion Bucket
- Floor Level, Area (SQFT)
- Sale Year, Sale Month, Sale Quarter
- Property Age (Years)
- MRT Distance, Top School Distance (if enrichment run)

**Features intentionally excluded:**
- Type of Sale — redundant, each model is already sale-type-specific
- Purchaser Address Indicator — buyer residency, not a property trait
- GLS features — used for display context only, not model input

## How to Run Training

1. Place `Non-Landed Caveats.xlsx` in the same folder
2. Open Command Prompt and run:

```
python nmrk_train_model.py
```

Training produces **40 files** (8 models × 5 file types). Takes roughly 15–25 minutes.

After training, upload all 40 new model files to GitHub.

---

## How to Run the App Locally

```
streamlit run nmrk_valuation_app.py
```

## Google Sheets (Valuer Feedback)

The app can write valuer feedback directly to a Google Sheet. To enable this:

### 1 — Google Cloud Setup
- Go to [console.cloud.google.com](https://console.cloud.google.com)
- Enable **Google Sheets API** and **Google Drive API**
- Create a Service Account under IAM & Admin
- Download the JSON key file

### 2 — Share Your Google Sheet
- Create a blank Google Sheet
- Click Share → add the service account email → set as Editor

### 3 — Create `.streamlit/secrets.toml`
Create this file in your project folder (never upload to GitHub):

```toml
[gsheets]
spreadsheet_id = "YOUR_SPREADSHEET_ID_HERE"

[gcp_service_account]
type = "service_account"
project_id = "your-project-id"
private_key_id = "..."
private_key = "-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"
client_email = "your-account@project.iam.gserviceaccount.com"
client_id = "..."
auth_uri = "https://accounts.google.com/o/oauth2/auth"
token_uri = "https://oauth2.googleapis.com/token"
auth_provider_x509_cert_url = "https://www.googleapis.com/oauth2/v1/certs"
client_x509_cert_url = "https://www.googleapis.com/robot/v1/metadata/x509/..."
```

### 4 — Streamlit Cloud
Go to your app → Settings → Secrets → paste the same content → Save → Reboot app.

The Google Sheet header row is created automatically on first submission.

## Disclaimer

This tool provides indicative estimates only and does not constitute a formal valuation or professional advice. Results are generated by a machine learning model trained on historical URA caveat data and are intended for reference and research purposes only. A formal valuation by a licensed and registered valuer should always be obtained for any significant property transaction.

Data source: Urban Redevelopment Authority (URA) of Singapore.
